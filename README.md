## Checkers

Run locally

1. Make sure node modules are installed
    - `npm install`
2. Run the app
    - `npm run start`
3. Open your browser to `http://localhost:8080`
---
Build for deployment

1. Make sure node modules are installed
    - `npm install`
2. Build the app
    - `npm run build`
3. See `build` directory was created
---
Test

1. Make sure node modules are installed
    - `npm install`
2. Run the tests
    - `npm run test`
---

Tested on Chrome, Firefox, Safari, and Edge
